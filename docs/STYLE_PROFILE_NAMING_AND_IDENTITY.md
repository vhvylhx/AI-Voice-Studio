# Style Profile Naming and Identity

Style Profile ID la khoa bat bien dang `style_000001`.

Display name co the doi.

Rename Style Profile khong duoc doi:

- style_profile_id;
- folder;
- extraction_state;
- Voice link;
- Variant link;
- export/import compatibility.

Ten moi phai:

- khong rong;
- khong qua dai;
- khong chua control character.

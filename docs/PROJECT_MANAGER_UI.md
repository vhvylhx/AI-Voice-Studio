# Project Manager UI

Project Manager là trang `Dự án`.

Trang hiển thị:

- Project hiện tại;
- danh sách Project active;
- danh sách Project archived;
- Project ID;
- display name;
- health;
- status;
- root path.

Action hiện có:

- Tạo;
- Mở;
- Đổi tên;
- Nhân bản;
- Archive;
- Restore Archive;
- Backup;
- Export;
- Validate;
- Xem folder.

Không có Delete vĩnh viễn trong Sprint này.

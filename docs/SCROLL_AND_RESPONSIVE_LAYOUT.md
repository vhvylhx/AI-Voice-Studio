# Scroll and Responsive Layout

Page dai phai dung scroll foundation thay vi ep MainWindow giu kich thuoc lon.

Component dung chung:

- `ContentScrollArea`
- `ScrollablePage`

Yeu cau:

- `setWidgetResizable(True)`;
- khong can horizontal scrollbar mac dinh;
- action cuoi trang truy cap duoc bang scroll;
- page co the mo o kich thuoc nho ma khong crash.

AVS-014.12 ap dung truoc cho TrainingPage vi day la page dai nhat sau khi tach workflow/reference.
